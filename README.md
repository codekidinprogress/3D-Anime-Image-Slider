# 🎴 3D Anime Image Slider

A stylish 3D image slider created using pure HTML and CSS. This project demonstrates how to use transforms, perspective, and animations to create a rotating gallery effect.

## ✨ Features
- 3D rotating image slider
- Smooth CSS animations
- Anime-themed design
- Responsive layout
- No JavaScript required

## 🛠️ Technologies Used
- HTML5
- CSS3 (Transforms, Animations, Flexbox)

## 📂 Project Structure
├── Slider.html
├── style.css
├── images/


## 🚀 How to Run
1. Download or clone the repository
2. Open `Slider.html` in your browser
